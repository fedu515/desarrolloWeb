Etiquetas de Texto



Nos permite utilizar y dar formato a texto.
  • p: define el texto
  • b:Nos permite poner un texto en negrita
  • i: nos permite poner un texto en cursiva
  • u nos permite sobrayar un texto
  • mark nos permite remarcar el texto
  • br nos permite realizar un salto de linea
  • hr nos permite dibujar una linea para demarcar sectores



LISTAS

Te permite estructurar contenido de las paginas, pueden crear filas con informacion y marcarlas con viñetas o numeros.
1- Listas ordenadas(Order List): Los elementos estan enumerados de 1..n. La etiqueta es < ol>.

  1. Verano
  2. Otoño
  3. Invierno
  4. Primavera


2 - Listas desordenadas(Unorder List): Los elementos de la lista se ordenan o enumeran con viñetas.
  • Fideo
  • Arroz
  • Verduras
  • Limpieza



TABLAS

Las tablas nos permiten tabular datos. Antes estas servian para maquetar una pagina web
La etiqueta que se usa para crear una tabla es:

  1. table: define una tabla
  2. tr: definen las filas de la tabla(table row)
  3. td: celdas de una tabla(table data)
  4. th: filas de encabezado dentro de la primera fila de la tabla, suele usarse para el titulo de la tabla (table header)

Enlaces

Nos permite enlazar documentos html entre sí, podemos enlazar secciones internas del propio documento o vincular sitios y aplicaciones externas 
                        La etiqueta que vamos a utilizar para crear un enlace es la etiqueta < a>.
                        < a href="">< /a>
                        href= Hace refrencia a la url del enlace que queremos vincular

Mas info sobre enlaces

Enlazar secciones internas de un mismo documento

                        Introducimos el concepto de ID que mas adelante vamos a explayar, este concepto hacer referencia a un identificador
                        unico dentro del documento. Para poder indicar a que ID deseo dirigirme, bastará con colocar un "#NombreDelID"
                        dentro del href.
                        Ejemplo:
                        < a href="#NombreDelID">

Etiquetas Semanticas

  • div: Etiqueta contenedora
  • header: Representa la cabecerea del contenido principal
  • nav: Representa un menu de navegación
  • main: Sección del contenido principal
  • aside: Representa una seccion de contenido complementario, se utiliza mayormente para poner una publicidad
  • article: Representa una serccion de autocontenido (que por si sola se explica)
  • Section: Representa una seccion de conetido generico
  • footer: Representa el pie de pagina
